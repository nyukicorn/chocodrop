🍫 ChocoDrop

話すだけで、3Dが生まれる

Three.jsに「ちょこんと」AIコンテンツを置くだけ。元のシーンを汚さず、気軽に創作実験。

✨ デモを試す 📦 GitHub

気軽に試せる設計

既存のシーンは絶対に変更されません。何度でも気軽に「ちょこっと置いて」試せます。

npm install github:nyukicorn/chocodrop

作る・置く・動かす・消す

import { createChocoDrop } from 'chocodrop';

const chocoDrop = createChocoDrop(scene, {
  camera,
  renderer
});

特徴

🛡️

非破壊実験

元のシーンを絶対に汚しません。いくらでも「ちょこんと置いて」安心して試せます。

🎭

作る・読む両対応

AI生成でも既存素材でも自由自在。「猫を作って」「動画を読み込んで」どちらもOK。

💬

自然な会話操作

「右上に」「ピンクにして」「大きくして」「消して」普通に話すだけで思い通りに。

🎨

創造性を解放する設計

シーンを汚さないから何度でも試せる。クリエイティブの自由度が格段にアップします。

こんな体験ができます

「炎を吐く赤いドラゴンの画像を作って」
→ AI画像生成で瞬時に配置
「猫がボールで遊んでいる動画を作って」
→ AI動画生成で動くコンテンツ
「ピンクに変更して」
→ 思いついたら即座に変更OK
「ここに移動して」
→ 言語でも手でも自由に配置
「気に入らないから削除して」
→ 失敗を恐れず何度でも実験
「全部終わったらリロード」
→ 元のシーンが綺麗に復活
