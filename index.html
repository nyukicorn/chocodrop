実際の使い方

1 空の3D空間で「うーん、ここの景色...」
2 "夕焼けの山" → ポンと配置
3 「違うな」→ Delete → "都市の夜景"
4 「これいい！」→ スクショ → 本実装へ

リロードしたら全部消える。それがいい。

導入は1行だけ

import { createChocoDrop } from '@chocodrop/core';

// あなたの既存のThree.jsシーン
const scene = new THREE.Scene();

// ChocoDrop追加（これだけ！）
const chocoDrop = createChocoDrop(scene, {
  camera,
  renderer
});

3つの特徴

🍫

ちょこっと配置、すぐ消せる

🌟 安心の実験設計

✨ 元ファイル保護 - インポートした画像・動画は絶対に変更されません
✨ シーン内編集のみ - 「ピンクにしたら？」「オーロラエフェクトは？」気軽に実験
✨ 作品は保存、編集は実験 - AIで生成したコンテンツはしっかり保存、色変更などはシーン内でのみ有効
✨ クリーンスレート - リロードで元の状態、「ちょっと試してみたい」を何度でも

🎨
