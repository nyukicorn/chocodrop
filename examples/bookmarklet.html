🍫 ChocoDrop Bookmarklet

ChocoDrop をどんな Three.js ページにも1クリックで追加できます。

📌 使い方

Step 1: daemon を起動 
npx chocodropd
Step 2: このリンクをブックマークバーにドラッグ＆ドロップ
🍫 ChocoDrop
Step 3: Three.js ページでブックマークをクリック

📝 コード

手動で使う場合は、このコードをコンソールに貼り付けてください：

(() => {
  const u = 'http://127.0.0.1:43110/sdk.js';
  if (document.getElementById('__chocodrop_sdk')) return;

  const s = document.createElement('script');
  s.id = '__chocodrop_sdk';
  s.src = u;

  s.onload = async () => {
    try {
      await window.chocodrop.ready();

      // Try to find scene object
      const scene = window.scene ||
                   document.querySelector('canvas')?.__scene__;

      if (scene) {
        await window.chocodrop.attach(scene, {});
      } else {
        console.warn('No Three.js scene found');
      }
    } catch (e) {
      console.error('ChocoDrop:', e);
    }
  };

  document.head.appendChild(s);
})();

🔧 注意事項

  • daemon が起動していることを確認してください
  • CORS のため、ローカルサーバーで配信されているページで使用してください
  • file:// プロトコルでは動作しません
